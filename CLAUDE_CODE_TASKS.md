## ЗАДАЧА 1 — Создать базовый шаблон посадочной страницы

**Файл:** `src/layouts/LandingLayout.astro`

Создай layout для всех SEO-посадочных страниц. Он должен принимать props:

```ts
interface Props {
  title: string          // <title> тег
  description: string    // meta description
  h1: string            // главный заголовок страницы
  canonical?: string     // canonical URL (необязательно)
}
```

**Структура layout:**
1. `<head>` — title, description, canonical, og:title, og:description
2. Тот же header что на главной (импортируй компонент Header)
3. `<main>` со слотом для контента
4. Тот же footer что на главной (импортируй компонент Footer)

**Важно:** Layout должен добавлять JSON-LD разметку Schema.org типа `EducationalOrganization` + `LocalBusiness`:

```json
{
  "@context": "https://schema.org",
  "@type": ["EducationalOrganization", "LocalBusiness"],
  "name": "АйДаКемп",
  "description": "IT-лагерь для детей 7–14 лет в Подмосковье",
  "url": "https://aidacamp.ru",
  "telephone": "+74951284429",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Наро-Фоминский район",
    "addressRegion": "Московская область",
    "addressCountry": "RU"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": 55.265643,
    "longitude": 36.724185
  }
}
```

## ЗАДАЧА 2 — Создать компонент LandingHero

**Файл:** `src/components/LandingHero.astro`

Этот компонент — адаптация текущего Hero под посадочные страницы. Принимает props:

```ts
interface Props {
  h1: string              // главный заголовок
  subtitle: string        // подзаголовок
  breadcrumb: string      // текст хлебной крошки, например "Лагерь программирования"
  keywords: string[]      // список ключевых фич (буллиты под заголовком)
}
```

**Структура компонента:**

```
[Хлебная крошка: Главная → {breadcrumb}]

H1: {h1}

{subtitle}

• {keywords[0]}
• {keywords[1]}  
• {keywords[2]}

[CTA кнопка "Узнать о свободных местах" → якорь на форму]
[Вторая кнопка "Смотреть смены" → якорь на блок смен]

[Мета-строка: 66 км от Москвы · смены июнь–август · от 48 000 ₽]
```

Хлебная крошка должна содержать Schema.org разметку BreadcrumbList:
```json
{"@type":"BreadcrumbList","itemListElement":[
  {"@type":"ListItem","position":1,"name":"Главная","item":"https://aidacamp.ru/"},
  {"@type":"ListItem","position":2,"name":"{breadcrumb}"}
]}
```

Дизайн: точно такой же стиль как на текущей главной (цвета, шрифты, Tailwind классы).

## ЗАДАЧА 3 — Создать компонент LandingContent

**Файл:** `src/components/LandingContent.astro`

Универсальный блок основного контента посадочной страницы.

Принимает props:
```ts
interface Props {
  sections: Array<{
    h2: string
    text: string
    list?: string[]   // необязательный список пунктов
  }>
}
```

Рендерит: чередующиеся H2 + параграф текста + опционально маркированный список.
Стиль: prose, читаемые отступы, тот же Tailwind что на сайте.

## ЗАДАЧА 8 — Обновить sitemap и robots.txt

### 8.1 sitemap
Убедиться что `astro.config.mjs` включает `@astrojs/sitemap` и все новые страницы попадают в sitemap.

Если sitemap не настроен:
```js
// astro.config.mjs
import sitemap from '@astrojs/sitemap'

export default defineConfig({
  site: 'https://aidacamp.ru',
  integrations: [sitemap()],
})
```

Установить если нет: `npm install @astrojs/sitemap`

### 8.2 robots.txt
Проверить что `public/robots.txt` содержит:
```
User-agent: *
Allow: /

Sitemap: https://aidacamp.ru/sitemap-index.xml
```
